bootstrap-select: dropdown selection buttons

jQuery plugin that leverages Bootrap's dropdown.js

I like these elements for a few reasons:

  • Responsive-friendly
    • The top-level element is compact and styles easily on mobile. The vertical drop down may be easily limited to a certain number of options with scrollable overflow. The horizontal form-factor may be fixed or option-text driven.
  • UX-friendly. (Nice for human people.)
    • The subtext feature allows for a secondary text attribute to be associated/rendered with the primary selection. This would be great, for example, for telegraphing how many records are associated with a given option. The dividers allow for visual cognitive breaks amongst higher categories of options. The menus pop up or down depending upon proximity to viewport boundary. Inline glyph icons are easily added for evocative, compact messaging.
  • Scalability
    • Options may be programmatically added in circumstances where an element of discovery is at play. Maybe the number of options varies depending upon the situation. These elements allow you to keep the top-level form-factor fixed (which is good for screen real estate planning) and yet give you managable scalability in the pop-down vertical dimension.
